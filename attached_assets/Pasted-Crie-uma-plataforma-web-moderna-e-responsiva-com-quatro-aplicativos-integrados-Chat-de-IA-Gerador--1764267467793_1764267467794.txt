Crie uma plataforma web moderna e responsiva com quatro aplicativos integrados: Chat de IA, Gerador de Prompt, Geração de Imagens e Geração de Vídeos. Entregue código pronto para rodar no Replit, com instruções claras de execução.

Visão geral da plataforma:
- Objetivo: Uma única aplicação com navegação entre 4 módulos: Chat, Gerar Prompt, Gerar Imagem, Gerar Vídeo.
- Arquitetura: Frontend SPA + Backend REST/streaming API.
- Tecnologias preferidas: 
  - Frontend: React + Vite (ou Next.js), Tailwind CSS.
  - Backend: Node.js + Express, WebSocket/SSE para streaming no chat.
  - Banco: SQLite simples via Prisma (ou JSON/local storage para MVP).
- Padrões: TypeScript, ESLint/Prettier, .env para chaves.
- Interface em português (pt-BR).

Funcionalidades por módulo:

1. Chat de IA
- Entrada de texto e resposta da IA com streaming.
- Histórico de conversas por sessão.
- Botão para ligar/desligar uso de contexto.
- Exportar conversa em .txt ou .md.
- Limpar conversa atual.
- Renderizar markdown (código, listas, negrito).
- API: /api/chat (stream SSE), /api/chat/sessions, /api/chat/session/:id.

2. Gerador de Prompt
- Campos: Objetivo, público-alvo, tom/estilo, restrições, formato, exemplos.
- Templates: coding, marketing, design, pesquisa.
- Botão “Refinar” para reescrever com mais clareza.
- Score de qualidade do prompt.
- Copiar para clipboard e salvar biblioteca.
- API: /api/prompt/generate, /api/prompt/refine, /api/prompt/save.

3. Geração de Imagem
- Campos: Prompt, estilo (realista, anime, minimalista, 3D), tamanho, número de variações.
- Seed opcional para reprodutibilidade.
- Variar imagem existente e fazer upscale.
- Galeria com preview, baixar PNG/JPG.
- API: /api/image/generate, /api/image/variate, /api/image/upscale.

4. Geração de Vídeo
- Campos: Prompt, duração, resolução (720p/1080p), estilo (cinemático, anime, abstrato), áudio, legenda.
- Storyboard opcional com 3–5 cenas.
- Preview em player, baixar MP4/WebM.
- API: /api/video/generate, /api/video/list, /api/video/:id.

Requisitos não funcionais:
- Autenticação com JWT.
- Rate limit por IP/usuário.
- Logs e métricas básicas.
- Mensagens de erro claras.
- Privacidade: variáveis de ambiente para chaves.
- Acessibilidade: WAI-ARIA, contraste adequado.

Interface e navegação:
- Header com logo “Plataforma IA”, menu com 4 apps.
- Página inicial com cards dos 4 apps.
- Tema claro/escuro com toggle.
- Componentes compartilhados: PromptEditor, ParamPanel, JobStatus, AssetCard.

Estrutura de pastas:
- /frontend: React + Vite, páginas e componentes.
- /backend: Express, rotas /api, provedores, fila de jobs.
- /shared: Tipos TypeScript e utilitários.
- /docs: README com instruções.

Variáveis de ambiente (.env):
- LLM_API_KEY
- IMAGE_API_KEY
- VIDEO_API_KEY
- JWT_SECRET
- DATABASE_URL

Testes:
- Unitários para funções de prompt.
- Integração para rotas /api/chat, /api/image, /api/video.
- Smoke tests para build e inicialização.
